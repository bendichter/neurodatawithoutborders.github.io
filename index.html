Neurodata Without Borders

Neurophysiology File Format

The Neurodata Without Borders: Neurophysiology ("NWB") format is designed to store a variety of neurophysiology data, including from intracellular and extracellular electrophysiology experiments, optical physiology experiments, as well as tracking and stimulus data.

This GitHub page is primarily used to organize pages related to source repository of the Neurodata Without Borders organization. For further information about NWB see:

  • Main Project Site http://www.nwb.org/nwb-neurophysiology/
  • General Overview : http://nwb-overview.readthedocs.io
  • Specification Language: http://schema-language.readthedocs.io
  • Format Specification: http://nwb-schema.readthedocs.io
  • Data Storage: http://nwb-storage.readthedocs.io
  • PyNWB (API): http://pynwb.readthedocs.io
  • MatNWB (API):https://github.com/NeurodataWithoutBorders/matnwb
